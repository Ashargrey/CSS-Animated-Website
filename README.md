# 🚀 Scroll Animation Website (HTML + CSS Only)

A modern and smooth **scroll-based animation website** built using only **HTML & CSS** — no JavaScript used!

## ✨ Features

- 🔥 Smooth scroll animations
- 🎯 Left-to-right & right-to-left box animations
- 💡 Clean and minimal UI
- ⚡ Lightweight (No JS required)
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3 (Animations & Transitions)
